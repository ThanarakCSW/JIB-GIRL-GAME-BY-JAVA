# � Jib-Girl Game (My Java Dating Sim)

ยินดีต้อนรับสู่ **Jib-Girl Game** เกมจำลองการจีบสาว (Dating Simulator) ที่สร้างขึ้นด้วยภาษา Java (Swing) แบบ 100%! ผู้เล่นจะต้องบริหารจัดการ **เวลา (Stamina)** และ **เงิน (Money)** เพื่อจีบสาวที่ตัวเองชอบให้ติดภายในเวลา 5 วัน

---

## ✨ ฟีเจอร์หลัก (Features)

### 🎮 โหมดผู้เล่นคนเดียว (Story Mode)
- **ระบบเนื้อเรื่อง 5 วัน:** ผู้เล่นต้องเลือกตัดสินใจในแต่ละเหตุการณ์ ซึ่งจะส่งผลต่อค่าความสัมพันธ์ (Relationship)
- **ฉากจบหลายแบบ (Multi-Endings):** มีทั้งจบแบบ Happy Ending (สมหวัง) และ Bad Ending (นก) ขึ้นอยู่กับคะแนนความรักที่คุณทำสะสมมา
- **ระบบบริหารจัดการทรัพยากร:**
  - **Stamina (พลังงาน):** ทุกการกระทำใช้พลังงาน ถ้าพลังงานหมดจะถูกบังคับข้ามวัน 
  - **Money (เงิน):** สามารถหาเงินได้จากการทำงานพิเศษ และใช้ซื้อของขวัญได้

### 🌐 โหมดเล่นหลายคนออนไลน์ (Online Multiplayer / Tournament)
- **Client-Server Architecture:** สร้างห้องเล่นกับเพื่อนผ่าน Socket Programming (รองรับผู้เล่นสูงสุด 3 คน)
- **Lobby Room:** หน้าจอรอผู้เล่นคนอื่น สามารถเช็คสถานะว่าใครพร้อมแล้ว และเลือกนางเอกที่จะจีบแบบไม่ซ้ำกันได้
- **Real-time Synchronization:** ดึงข้อมูลคะแนนและเวลาของผู้เล่นคนอื่นในห้องมาโชว์บนหน้าจอระหว่างเล่นได้!
- **ซิงค์ตอนจบ (Synchronized Game End):** ระบบจะรอให้ทุกคนเล่นจบก่อน จึงจะแสดงหน้าสรุปผล (Leaderboard) แบบยุติธรรม
- **ระบบจัดอันดับ & เสมอ (Tie Handling):** ใครคะแนนสูงสุดจะได้เป็น **"พระเอก"** (การ์ดสีทอง) หากคะแนนเท่ากันก็ถือว่าชนะร่วมกัน!

### 🛍️ ระบบร้านค้าและกระเป๋า (Shop & Inventory)
- **ร้านค้าน่ารักสุดคิวท์:** ซื้อของขวัญหลากหลายระดับราคา (ตั้งแต่ 45฿ ไปจนถึง 800฿)
- **กระเป๋าช่องเก็บของ:** เก็บของที่ซื้อไว้เพื่อนำไปมอบให้ตัวละคร
- **ระบบความชอบส่วนตัว (Character Preferences):**
  - ของขวัญราคาแพงจะได้คะแนนเนื้อเรื่องเยอะกว่า (เช่น ของราคา 500+ ได้ความรัก +30)
  - ถ้าให้ของขวัญตรงกับ **"ของโปรด"** ของตัวละครนั้นๆ จะได้โบนัสคะแนนพิเศษเพิ่ม! (+5)

### 🌸 3 ตัวละคร 3 สไตล์
1. **มะปราง (Maprang):** สาวศิลปิน อาร์ตตัวแม่ สุขุม นุ่มลึก
2. **ไอซ์ (Ice):** สาวนักกีฬา ห้าวๆ ลุยๆ พลังงานล้นเหลือ
3. **ขนม (Khanom):** สาวน้อยน่ารัก สดใส พลังบวกเต็มเปี่ยม

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Language:** Java 25 (รองรับเวอร์ชั่นเก่ากว่าที่รัน Swing ได้)
- **GUI Framework:** Java Swing / AWT
- **Network:** Java Sockets (`java.net.Socket`, `java.net.ServerSocket`)
- **UI Design:** Custom Components (วาด UI ขึ้นมาเอง เช่น `PremiumButton`, `ModernPanel`, `CuteCard` เพื่อให้หลุดจากกรอบ UI เดิมๆ ของ Swing)
- **Concurrency:** ใช้ Threads และ `SwingWorker` ในการจัดการ Network สลับกับอัปเดตหน้าจอ UI

## 🚀 วิธีการเข้าเล่นเกม (How to Play)

### 1. การคอมไพล์และรันโปรแกรม
เนื่องจากเป็นโปรเจกต์ Java มาตรฐาน ให้ชี้ Main Class ไปที่:
```bash
com.jibgirl.main.Main
```

### 2. วิธีการรันโหมด Online Multiplayer
ต้องเปิดโปรแกรมอย่างน้อย 2 หน้าต่าง (1 Server, 1+ Clients)
1. **เปิด Server (Host):**
   - รันโปรแกรมหลัก
   - ที่หน้า Start Screen เลือกแถบ `[ ผู้พัฒนา ]` (มุมล่างขวา) -> กดปุ่ม **"เปิดเซิร์ฟเวอร์"** (ระบบจะรัน Server ไว้ที่ Port `12345`)
2. **เปิด Client (ผู้เล่น):**
   - รันโปรแกรมหลัก (เปิดจำลองเครื่องเพื่อน)
   - หน้า Start Screen กดปุ่ม **"เล่นโหมด ONLINE 🌐"**
   - กรอกชื่อผู้เล่น (ถ้าใช้คอมเดียวกัน ให้ใส่ที่อยู่ IP เป็น `localhost` หรือ `127.0.0.1`)

## 📁 โครงสร้างโปรเจกต์ที่น่าสนใจ (Project Structure)
- `com.jibgirl.main`: จุดเริ่มรันโปแกรม (`Main.java`)
- `com.jibgirl.network`: ระบบเครือข่ายทั้งหมด (`GameServer`, `GameClient`, `GameResult`)
- `com.jibgirl.view`: หน้าจอ GUI ทั้งหมด (`GameGui`, `LobbyScreen`, `OnlineEndingScreen`, `StartScreen`, `ShopUI`, `ModernInventoryUI`)
- `com.jibgirl.Day`: คลาสเนื้อเรื่องแต่ละเส้นทางและตัวเลือกของแต่ละวัน
- `com.jibgirl.utils`: คลาสช่วยเหลือเช่น ตัวจับเวลา (`QuestionTimer`) และ UI Components สไตล์น่ารัก (`UIUtils`)
